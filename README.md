C-Tester
========

C-Tester is a platform for running small applications (games, questionaries, testing apps) on Web and mobile devices with Android. 

### Requirements
 * Cordova (6.4.0.)
 * Python 2.7+ (tools, scaffolding)
   * packages Jinja2, pyaml
